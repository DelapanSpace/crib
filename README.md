## Project Name: Crib of delapan space

**crib** is a website for my creative agency and serves as the digital home of **Delapan Space**.  
It represents our approach to creative work, storytelling, and digital presence, acting as a central hub for showcasing identity, values, and ongoing initiatives.

The project is built with a modern web stack to ensure performance, scalability, and long-term maintainability.

---

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Package Manager**: pnpm
- **CMS**: Sanity.io (Headless CMS)
- **Styling**: Modern CSS / utility-first approach
- **Deployment**: Vercel or self-hosted (VPS compatible)
